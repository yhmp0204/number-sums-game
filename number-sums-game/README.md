# Number Sums

数字を選んで合計を目標値に合わせるパズルゲームです。

![Number Sums Screenshot](screenshot.png)

## 🎮 遊び方

1. グリッド内の数字をタップして選択（⭕）または消去
2. 行・列・色付きエリアの合計を目標値に合わせる
3. 間違えるとライフが減る（2回でゲームオーバー）
4. すべての行・列・エリアを正しく完成させるとクリア！

## ✨ 特徴

- **3段階の難易度**: 簡単(7×7)、普通(8×8)、難しい(9×9)
- **直感的な操作**: タップで選択/消去を切り替え
- **視覚的フィードバック**: 
  - 現在の合計値をリアルタイム表示
  - オーバーフロー時は赤色で警告
  - エリア完成時は白抜きで表示
- **モバイル対応**: スマートフォンで快適にプレイ可能
