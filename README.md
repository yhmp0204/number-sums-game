# Number Sums

数字を選んで合計を目標値に合わせるパズルゲームです。

## 🎮 遊び方

1. グリッド内の数字をタップして選択（⭕）または消去
2. 行・列・色付きエリアの合計を目標値に合わせる
3. 間違えるとライフが減る（2回でゲームオーバー）
4. すべての行・列・エリアを正しく完成させるとクリア！

## ✨ 特徴

- **3段階の難易度**: 簡単(7×7)、普通(8×8)、難しい(9×9)
- **2,500問の固定パズル**: 各問題は一意な解を持つ
- **進捗保存**: LocalStorageでクリア状況を記録
- **ライフシステム**: 5ライフ、30分ごとに1回復
- **タイマー機能**: クリアタイムを記録
- **星評価**: ミス回数に応じて1〜3つ星

## 🔧 技術的特徴

### シード値による問題固定
- 問題番号からシード値を生成
- 同じ問題番号は常に同じパズルを生成
- 全ての問題は一意な解を持つことを保証

### 解の一意性検証
- バックトラッキングアルゴリズムで全解を探索
- 2つ目の解が見つかった時点で再生成
- 枝刈り最適化で高速化
